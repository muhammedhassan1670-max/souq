# أول تشغيل للوحة تحكم سوق البلد

اتبع الخطوات دي بعد تشغيل Supabase وربط `.env`.

## 1. تسجيل الدخول

افتح:

```text
/admin/login
```

استخدم البريد وكلمة المرور التي أنشأتها من Supabase Auth.

## 2. إضافة أول قسم

افتح `/admin/categories`.

أضف مثلًا:

- الاسم: بقالة
- Icon: ShoppingBag
- نشط: نعم

اضغط حفظ.

## 3. إضافة أول محل

افتح `/admin/shops`.

أضف:

- اسم المحل
- اسم المالك
- رقم الموبايل
- نوع المحل
- العنوان
- صورة المحل إن وجدت

اضغط حفظ.

## 4. إضافة أول منتج

افتح `/admin/products`.

املأ:

- اسم المنتج
- القسم
- المحل
- السعر
- الوحدة
- الصورة
- متاح
- كلمات البحث

اضغط حفظ، ثم افتح `/products` وتأكد أن المنتج ظهر للعميل.

## 5. تغيير رقم واتساب

افتح `/admin/settings`.

غيّر:

- رقم الواتساب
- رقم الاتصال

اضغط حفظ، ثم جرّب زر واتساب من صفحة العميل.

## 6. تغيير رسوم التوصيل

من `/admin/settings` غيّر:

- رسوم التوصيل
- حد التوصيل المجاني

اضغط حفظ، ثم جرّب Checkout.

## 7. تجربة طلب من صفحة العميل

1. افتح الصفحة الرئيسية.
2. أضف منتجين للسلة.
3. اضغط `كمّل الطلب`.
4. اكتب الاسم والموبايل والعنوان.
5. اضغط `ابعت الطلب واتساب`.
6. تأكد أن الطلب ظهر في `/admin/orders`.
7. غيّر حالة الطلب من لوحة التحكم.
